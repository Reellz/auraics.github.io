# Aura International Christian School Website

Aura International Christian School is a modern, interactive website built primarily with HTML, CSS and JavaScript on the backend and standard web technologies on the frontend. The project aims to deliver a seamless and engaging user experience through clean design, responsive layouts, and dynamic content.

## 🚀 Live Demo

You can explore the live version of the app here:  
[Aura International Christian School Live Site](https://auraics.vercel.app)

Feel free to navigate through the pages and test the interactive features.

## 🛠️ Technologies Used

- **Frontend:**  
  - HTML5 for semantic structure  
  - CSS3 for styling and responsive design  
  - JavaScript for interactivity, frontend and backend logic  
- **Form Submission:**  
  - [FormSpree Ajax](https://formspree.io)
  - [Web3Forms](https://web3forms.com) 
- **Deployment & Hosting:**  
  - Hosted on [Vercel](https://vercel.com) for fast and reliable delivery

## 📁 Project Structure

Here’s an overview of the main files and directories in this repository:

- `index.html`  
  The main entry point and homepage of the application.

- `pages/`  
  Contains additional HTML pages that extend the website’s content and functionality.

- `styles/`  
  All CSS stylesheets are organized here to keep the design consistent and maintainable.

- `images/`  
  Folder for all image assets used throughout the site including icons, backgrounds, and logos.

- JavaScript files:  
  - `script.js`: Handles core frontend logic and interactions.  
  - `admissions.js`, `contact.js`, `events.js`, `form.js`, `thank-you.js`: Modular scripts targeting specific pages or features to keep the code organized.

- `.vscode/`  
  Configuration files for Visual Studio Code, useful for consistent editor settings across contributors.

- `LICENSE`  
  The license file outlining the terms under which the code can be used and distributed.
